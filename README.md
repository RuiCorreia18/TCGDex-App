# TCGDex-App

You are required to write an Android application that uses TCGDex Api
(https://tcgdex.dev/).
The application should follow these rules
• You are not allowed to use the TCGDex SDK, only the cURL
• You can use any other library
• The application should have two screens
1. A screen with a list of Pokémon cards with the option to search by name
2. A screen showing the details of a card when selected
